Resolve Memories

Using Healing Pathways That Are Already Inside You

This website is a self-guided introduction to a specific way to do Flash* approaches to quickly resolve difficult memories one memory or experience at a time. This approach is a way to process memories with minimal distress. If you encounter distress in any part of this work, immediately stop and deal with the distress using the resources introduced below.

This healing pathway involves steps and procedures that are not intuitive. Do the preparation work below to increase the chance of having a positive experience. The authors of this website are not responsible for any potential harm you may encounter from the use or misuse of this tool.
